## ✨ 这是什么

浮云是一个运行在 LSPosed 框架下的微信辅助模块。当你把某些联系人标记为"密友"后，浮云会在微信运行时把与这些联系人相关的痕迹从界面上抹去，让别人拿到你手机、翻看你微信时，看不到这些人的存在。

---

## 🧩 核心功能

| 功能 | 说明 |
|------|------|
| 🗂️ **会话列表隐藏** | 主页聊天列表中隐藏密友的会话 |
| 🚫 **禁止进入聊天** | 拦截打开密友聊天窗口的操作 |
| 📞 **拦截 VoIP 来电** | 屏蔽密友发起的语音/视频通话来电界面 |
| 👥 **通讯录隐藏** | 通讯录列表中隐藏密友条目 |
| 🪪 **禁止查看资料页** | 拦截进入密友联系人资料页 |
| 🌐 **朋友圈隐藏** | 隐藏朋友圈入口相关内容及指定动态 |
| ➕ **长按快速添加密友** | 在联系人上长按即可将其加入密友名单 |
| 🔓 **临时解除隐藏** | 通过标题多击 / 长按临时显示被隐藏的内容 |
| ⚡ **原生流畅度** | 列表滑动与返回动画期间零开销放行，保持微信原生手感 |

---

## 🔧 兼容性与框架

- **支持**：当前适配目标 **8.0.74**未来更新频率：为了避免热更新导致插件功能失效，最高只会适配最新版的前一版微信
- **框架**：仅支持 **LSPosed 102 API** 框架
